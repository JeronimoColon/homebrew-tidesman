# homebrew-tidesman

Homebrew tap for [Tidesman](https://tidesman.dev), a native MCP server for running, understanding,
and debugging Linux containers on macOS with Apple's `container` tool.

```sh
brew install JeronimoColon/tidesman/tidesman
```

Requires an Apple Silicon Mac on macOS 26 with Apple's `container` installed and started
(`container system start`). Downloads and full documentation:
<https://github.com/JeronimoColon/tidesman-mcp>.
